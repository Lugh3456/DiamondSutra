# Diamond Sutra · 金刚般若波罗蜜经

A personal bilingual study site for the Diamond Sutra, one of nine scripture mini sites under [The Dharma Gate](https://lugh3456.github.io/DharmaGate/) portal.

## What it is

A chapter-by-chapter study companion for the Diamond Sutra (金刚经), divided into 32 chapters following the traditional Kumārajīva translation. Each chapter presents the full Chinese text with speech synthesis, followed by explanation cards with short Chinese explanations and expandable detailed English commentary.

## Features

- Bilingual Chinese/English throughout
- Default view shows Chinese explanations only (concept-focused, accessible register)
- Expandable "English ▾" toggle reveals English translation + full English commentary
- Bilingual summary at the end of each chapter (Chinese default, expandable English)
- Speech synthesis for full Chinese text (Web Speech API)
- Portal bar linking back to The Dharma Gate hub
- Mobile-optimised, responsive layout

## Tech stack

- Pure HTML + external CSS, no frameworks
- CSS design tokens (red/saffron colour scheme, shared with HeartSutra)
- Google Fonts: Noto Serif SC + Noto Sans SC
- No build step required

## Local preview

Open `index.html` directly in a browser.

Push all files to the root of the GitHub repository. GitHub Pages serves it automatically.
